# Extension Icons

Please add three icon files to this directory:

- **icon16.png** (16x16 pixels) - Toolbar icon
- **icon48.png** (48x48 pixels) - Extension management page
- **icon128.png** (128x128 pixels) - Chrome Web Store

You can use any meeting/note-taking themed icon. Quick options:
- Use an online icon generator (e.g., flaticon.com, icons8.com)
- Create simple icons with tools like Canva or Figma
- Use emoji-to-PNG converters (üìù notepad emoji)

Recommended colors: Blue (#4285F4) or Green (#34A853) to match Google Meet branding
